# Light Write
Simple Note App

Version 1.0

Features:
1. Changing theme :
      1.1 Black
      1.2 White
      1.3 Dark
   
2. Changing font size
      2.1 From 10 to 17
      2.2 By default: 11
   
3. Work with file:
      3.1 Open
      3.2 Save
      3.3 New File
   
4. Command Line
      4.1 Watch 2.Changing font size
